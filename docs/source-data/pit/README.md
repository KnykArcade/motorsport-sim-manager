# Pit transit source data

Place the authoritative workbook here:

```text
PIT_DATA_TRANSIT_ONLY_COMPLETED(1).xlsx
```

This workbook is the editable source of truth for pit entrance, pit-lane transit, and pit-exit loss. It must not be used to generate or infer stationary pit-stop service time.

Runtime data is generated from the workbook by `scripts/convertPitWorkbook.mjs` and loaded from `src/data/pit/`.

Locked accounting rule:

```text
totalPitVisitLossSeconds =
  transitLossSeconds
  + individualPitStopSeconds
  + queueDelaySeconds
  + entryExitErrorSeconds
  + penaltyDelaySeconds
  + repairDelaySeconds
  + cautionAdjustmentSeconds
```

`transitLossSeconds` comes from `Track_Pit_Lane.Pit_Transit_Loss_Sec` exactly once. `individualPitStopSeconds` is generated by each runtime pit-stop event and is never supplied by the workbook.
